## 技术架构
采用微服务架构，基于 Kubernetes 容器编排平台部署。
- 数据采集层：Kafka + Flink 实时流处理
- 数据存储层：MySQL + ClickHouse + MinIO
- 数据服务层：Spring Cloud + GraphQL
